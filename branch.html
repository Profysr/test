Branches Command

Hamari software development k process mein different teams hissa le ryii hoti hain to is tarha hr ak jo bhi change kare ga wo main repo mein commit karte rhee ga jis se code ko manage ko manage karna mushkil ho jye ga. isi lye hum hr team k lye alg alg Branches banate hain taa ke hr ak team apna kaam complete kare us code ko merge karne k request de de jis se second advantage yh hota hai k senior developer code ko peer review karleta hai

  1. git branch --- total no of branches or current branch k baare mei btati hai
  2. git branch -m newName --- to rename the branch
  3. git branch name --- ak new branch bana deti hai
  4. git branch -d name --- to delete any existing branch pr usee k lye humein ksii dosri branch pr navigate karna paddta hai
  5. git checkout toNavigateBranchName --- ak branch se dosri branch mein jaane k lye
  6. git checkout -b newName --- multiprocess cmd hai jis mei yh ak new branch bhi create kardeti hai or humein new branch mein navugate bhi kardeti hai
